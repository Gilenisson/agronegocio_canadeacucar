# Sistema de Controle de Perdas na Colheita de Cana-de-Açúcar

## Descrição do Projeto

Este projeto foi criado com o objetivo de desenvolver uma solução simples em Python para controle de perdas na colheita de cana-de-açúcar.

O agronegócio é um dos setores mais importantes da economia brasileira, e a produção de cana-de-açúcar se destaca como uma das principais atividades agrícolas do país. No entanto, durante a colheita, principalmente a mecanizada, ocorrem perdas que podem chegar a até 15% da produção.

Diante disso, este sistema foi desenvolvido para ajudar no registro e análise dessas perdas, auxiliando o produtor rural na tomada de decisão.

---

## Problema do Agronegócio

O problema tratado neste projeto é a perda de produtividade na colheita da cana-de-açúcar.

Essas perdas representam prejuízos financeiros e impactam diretamente a eficiência do setor agrícola. Muitas vezes, os produtores não possuem um controle detalhado dessas perdas, o que dificulta a melhoria do processo.

---

## Objetivo da Solução

- Registrar dados de colheita;
- Calcular a quantidade de perda em toneladas;
- Calcular o percentual de perda;
- Classificar a perda (baixa, moderada ou alta);
- Armazenar os dados em arquivo JSON;
- Gerar relatório em arquivo texto;
- Organizar as informações de forma clara.

---

## Conteúdos de Python Utilizados

### Subalgoritmos
O sistema utiliza funções com passagem de parâmetros, como:
- leitura de dados do usuário;
- cálculo de perdas;
- classificação de resultados.

### Estruturas de Dados
- Lista: armazenamento das colheitas;
- Tupla: classificação das perdas;
- Dicionário: estrutura de cada registro;
- Tabela de memória: representação das variáveis durante execução.

### Manipulação de Arquivos
- JSON: armazenamento dos dados;
- TXT: geração de relatórios.

### Banco de Dados (Oracle)
O sistema foi estruturado para futura integração com banco de dados Oracle, podendo armazenar os dados de forma mais robusta em versões futuras.

## Funcionalidades do Sistema

1. Cadastrar colheita
